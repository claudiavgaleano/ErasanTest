# Stellar - React + Vite + Material UI

A modern React application built with Vite and Material UI featuring a beautiful dark theme, responsive navigation, and a fully functional contact form.

## Features

- ⚡ **Vite** - Lightning fast build tool
- ⚛️ **React 18** - Modern React with hooks
- 🎨 **Material UI** - Beautiful UI components
- 🎯 **React Router** - Client-side routing
- 📱 **Responsive Design** - Mobile-first approach
- 🌙 **Dark Theme** - Elegant dark mode design
- ✨ **Animations** - Smooth transitions and effects
- 📝 **Contact Form** - With validation

## Pages

- **Home** - Landing page with hero section and features
- **About** - Company information and team
- **Services** - Service offerings
- **Contact** - Contact form with validation

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
   ```bash
   cd my-react-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:3000`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
